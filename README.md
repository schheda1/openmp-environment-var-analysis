# OpenMP (Host) Environment Variable Analysis
This repository contains the cleaned, processed and aggregated data used in the paper (https://doi.org/10.1109/SCW63240.2024.00131). 

File names follow the follwing convention: 
```
<cpu-architecture>-<benchmark-name>-<input-size OR thread count with default inputs>-agg.csv
```
